# Core Decisions

## Decision Records

### D-001: Language Choice
**Status**: Accepted  
**Date**: 2026-06-13

We choose Go as our primary language because:
- Excellent concurrency support
- Fast compilation and execution  
- Strong standard library
- Good performance characteristics for microservices

### D-002: Storage Approach
**Status**: Accepted  
**Date**: 2026-06-13

We use local file system storage via the MCP protocol because:
- Zero external dependencies
- Simple deployment
- Easy debugging and inspection
- Sufficient performance for this project scope
